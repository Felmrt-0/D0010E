In this package there is two classes, GameGrid and GomokuGameState. GameGrid reads locations of the grid, size of the grid, enters a move in the grid, clears the grid and checks if a player has won. GomokuGameState sets the amount of rows in the board. It also handles the messages, returns the game grid, and checks if a move is legal. It also handles the starting of a game, receiving a move from opponent, receving a game request, opponent disconnect and player disconnect.
